# Weather Hero

> AI-generated React weather app, built to practice CI pipelines and GitHub Actions.

## Stack

- React + TypeScript (Vite)
- [Open-Meteo](https://open-meteo.com/) — free, tokenless weather API
- Vitest + React Testing Library
- ESLint

## Getting started

```bash
npm install
npm run dev       # start dev server
npm test          # run unit tests
npm run lint      # run linter
npm run build     # build for production
```
